Deploy EduMaster to Netlify

This guide will help you deploy your EduMaster educational platform to Netlify.

Deployment Package Information

File: edumaster-deploy.zip

Location: Same directory as this file

Contents: All built files for your EduMaster application

Method 1: Drag and Drop Deployment (Easiest)

Step 1: Create a Netlify Account

Visit Netlify Signup to create a free account.

Step 2: Deploy Your Site

  1. Go to Netlify Dashboard
  2. Sign in to your account
  3. Drag and drop the edumaster-deploy.zip file onto the deployment area
  4. Netlify will automatically deploy your site

Step 3: Configure Your Site

  1. After deployment, click on your new site in the Netlify dashboard
  2. Your site URL will be displayed (e.g., https://your-site-name.netlify.app)
  3. You can customize the site name in "Site settings" > "Change site name"

Method 2: GitHub Integration (Recommended for automatic deployments)

Step 1: Push Your Code to GitHub

If you haven't already, push your code to a GitHub repository:

git add .
git commit -m "Prepare for Netlify deployment"
git remote add origin https://github.com/yourusername/your-repo-name.git
git push -u origin main

Step 2: Connect to Netlify

  1. Go to Netlify Dashboard
  2. Click "New site from Git"
  3. Select "GitHub" and authorize Netlify
  4. Select your EduMaster repository

Step 3: Configure Build Settings

Netlify should automatically detect your settings from netlify.toml:

  • Build command: npm install && npm run build
  • Publish directory: dist

Click "Deploy site"

Troubleshooting

Common Issues

  • Routing problems: Ensure redirect rules are applied
  • Asset loading: Check that base: './' is in vite.config.ts
  • Build failures: Verify Node.js version 20 in netlify.toml
